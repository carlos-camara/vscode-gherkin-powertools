# 🛠️ Command Center

Discover all extension capabilities from a single place!

The **Command Center** provides an interactive QuickPick menu where you can access:
- **🎨 Formatting:** Format documents instantly
- **▶️ Execution:** Run and Debug Features and Scenarios
- **🛠️ Step Definitions:** Auto-generate Python steps
- **📊 Analysis:** View Statistics and Diagnose Workspace

Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux) and search for **Gherkin PowerTools: Command Center**.
