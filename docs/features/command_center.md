# 🛠️ Command Center

Discover all extension capabilities from a single, unified interface!

## Overview

Memorizing keyboard shortcuts and discovering extension capabilities can be overwhelming. The **Command Center** provides an interactive QuickPick menu where you can access:

- **🎨 Formatting:** Format documents instantly
- **▶️ Execution:** Run and Debug Features and Scenarios
- **🛠️ Step Definitions:** Navigate to step definitions
- **📊 Analysis:** View Statistics and Diagnose Workspace
- **⚙️ Configuration:** Quickly access extension settings

## Usage

Press <kbd>Cmd+Shift+P</kbd> (Mac) or <kbd>Ctrl+Shift+P</kbd> (Windows/Linux) and search for **Gherkin PowerTools: Command Center**.
