# Security Policy

We take the security of **Gherkin PowerTools** seriously.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.5.x   | :white_check_mark: |
| < 1.5.0 | :x:                |

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please do **not** open a public issue. Instead:

1. **Contact**: Send a detailed report to the maintainer via email
2. **Information Required**: Description, reproduction steps (POC), impact assessment
3. **Acknowledgement**: Response within 48 hours
4. **Fixing**: Critical issues resolved within 7 days

## Security Notes

This extension operates entirely locally within VS Code and does not transmit any data to external servers.
